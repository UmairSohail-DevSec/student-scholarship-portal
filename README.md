# 🎓 Student Scholarship Portal (Management System)

This is a complete web-based system designed to help students find the right scholarships (like Global UGRAD and Chinese Government Scholarships) based on their education and marks.

---

## 🛠️ System Architecture & Workflow
The system is built on a modular architecture that ensures a seamless flow between data entry and user consumption:

1. **Authentication Layer:** Secure login/signup for students and a protected gateway for administrators.
2. **Logic Engine:** A backend algorithm that filters global opportunities (UGRAD, CSC, etc.) based on student academic credentials.
3. **Data Persistence:** Optimized MySQL database schema to handle relational data between users and scholarship records.

---

## 🚀 Core Features
### 👤 Student Experience
* **Personalized Dashboard:** Users can manage their academic profiles.
* **Smart Filtering:** Instant matching with scholarships based on Degree Level (Bachelors, Masters, PhD) and Marks.
* **Global Database:** Access to verified international education opportunities.

### 🔑 Administrative Control (CMS)
* **Full CRUD Lifecycle:** Admins can Create, Read, Update, and Delete scholarship records with zero downtime.
* **Dynamic Content Management:** Instantly update scholarship benefits, deadlines, and eligibility criteria.
* **User Oversight:** Ability to manage the system via a secure, dark-themed professional dashboard.
